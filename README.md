# robot-coordination-system

## To download the application:

1. `git clone git@github.com:waidfj/robot-coordination-system.git`
2. `cd robot-coordination-system.git`
3. `cargo build`
4. `cargo run`

## To add new work to the application:

1. `git checkout -b <branch-name>`
2. `git add .`
3. `git commit -m <commit-message>`
4. `git push`
5. Then head to https://github.com/waidfj/robot-coordination-system and create a pull request.
